Ez egy játék amit Pythonban fejlesztettem középiskolai szakdolgozatként. A játék egy 2D-s rage game, ami a Syobon Action-re (Közismert nevén Cat Mariora)  hasonlít.
A játék célja az, hogy kijátsszuk a pályákat. A pálya végén lévő zászlót kell megérinteni,
ám az oda vezető utunkat különféle akadályok nehezítik. 

Futtatás:
1. Töltsük le a Pythont a https://www.python.org/ weboldalról. Ajánlott telepítéskor hozzáadni a környezeti változókhoz (Add to PATH variable)
2. Nyissunk egy CLI-t (Terminal, PowerShell, Cmd stb)
3. Navigáljunk el a projekt mappájába oda, ahol a main.py fájl található (vagy akár direkt ide is nyithatjuk a CLI-t)
4. Készítsünk egy virtuális környezetet a "python -m venv .venv" paranccsal
5. Töltsük le a pygame-et a "pip install pygame" paranccsal ("-k nélkül a CLI-be írjuk bele)
6. Futtassuk a main.py fájlt a "python main.py" paranccsal  ("-k nélkül a CLI-be írjuk bele).

A program megírása során értettem meg az OOP, az importálható könyvtárak és modulok és a fájlkezelés működését igazán.
A stressz alatti, romlás nélküli teljesítményemet és a problémamegoldó képességeimet fejlesztette, ugyanis
a program megírása alatt nem használtam AI-t, csak dokumentációt és videókat, illetve időre kellett megírnom.

Használt technológiák:
- Python
- Pygame
